📚 Biblioteca CInForme

Um sistema de biblioteca em C++ com orientação a objetos, autenticação e tabela hash.

📌 Sobre o Projeto

O Biblioteca CInForme é um sistema de gerenciamento de biblioteca desenvolvido em C++. Ele permite o cadastro e login de usuários, empréstimo e devolução de livros, edição e exclusão de itens da biblioteca, utilizando estrutura de dados eficientes como vetor e tabela hash. O projeto foi pensado para praticar e aplicar os principais conceitos da Programação Orientada a Objetos (POO).

🧠 Conceitos de Programação Orientada a Objetos

  • Herança: A classe Livro herda de ItemBiblioteca.
  • Polimorfismo: Uso de método virtual puro exibirInformacoes() implementado por subclasses.
  • Encapsulamento: Atributos protegidos e privados acessados via getters/setters.
  • Classe Abstrata: ItemBiblioteca obriga implementação de métodos essenciais pelas classes derivadas.
  • Tabela Hash: Livros são armazenados de forma eficiente em uma tabela hash para acesso rápido por ID.

🛠️ Estrutura dos Arquivos

  • ItemBiblioteca.h / .cpp — Classe abstrata base.
  • Livro.h / .cpp — Subclasse de ItemBiblioteca.
  • Usuario.h / .cpp — Representa o usuário e seus livros alugados.
  • Biblioteca.h / .cpp — Controla o sistema: usuários, livros, login e ações.
  • main.cpp — Ponto de entrada. Gerencia os menus de interação.

🧾 Resumo das Funções

🔹 ItemBiblioteca

  • exibirInformacoes() – método abstrato a ser implementado nas subclasses.
  • getId(), getTitulo(), getAutor(), isAlugado() – métodos de acesso.
  • setTitulo(), setAutor(), setAlugado() – métodos modificadores.

🔹 Livro

  • Construtor: inicializa id, título e autor.
  • exibirInformacoes() – exibe detalhes do livro e status de aluguel.

🔹 Usuario

  • Construtor: define nome e senha.
  • autenticar() – valida a senha do usuário.
  • alugarLivro() – registra o ID do livro alugado.
  • devolverLivro() – remove o livro da lista de alugados.
  • getLivrosAlugados() – retorna todos os livros alugados.

🔹 Biblioteca

  • Cadastrar_Usuario() – adiciona um novo usuário.
  • Login() – autentica e armazena o usuário logado.
  • Logout() – encerra a sessão.
  • Adicionar_Livro() – cria e insere um livro na hash.
  • Listar_Livros() – exibe todos os livros da biblioteca.
  • buscarLivroPorId() – retorna o livro pelo ID.
  • hash() – função de espalhamento da tabela.
  • Alugar_Livro() – marca como alugado e vincula ao usuário.
  • Devolver_Livro() – torna o livro disponível novamente.
  • Listar_Livros_Alugados() – mostra livros alugados pelo usuário atual.
  • Editar_Livro() – altera título ou autor.
  • Remover_Livro() – exclui o livro da biblioteca.

🔹 Menu e Execução

  • main() – inicia o programa e gerencia o menu inicial.
  • MenuBiblioteca() – opções disponíveis após o login.
